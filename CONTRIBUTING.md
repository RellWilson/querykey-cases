# Contributing

Thanks for helping improve QueryKey Cases!

- Issues: use the Bug Report or Feature Request templates under New Issue.
- PRs: small, focused changes preferred; link an issue and describe the change.
- Style: keep examples concise; use YOUR_TOKEN for placeholders.
- DCO: sign commits with `-s` to certify the Developer Certificate of Origin.

By contributing, you agree your contributions are licensed under Apache-2.0 (code/schema) and CC BY 4.0 (docs).
